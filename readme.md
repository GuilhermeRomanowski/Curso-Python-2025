## Curso Introdutório de Python
Já utilizo python no meu trabalho a um tempo. Aprendi muita coisa na prática, resolvendo problemas reais. Agora inicio um curso do básico ao avançado com o objetivo de:

1. Organizar fundamentos.
2. Aprender os termos técnicos corretos.
3. Preencher lacunas que só aparecem quando a gente aprende sozinho.

Com este curso gratuito, pretendo entender não só os fundamentos, mas evoluir e continuar estudando algumas bibliotecas importantes (principalmente na área de dados)

## O que espero aprender neste curso

1. Aprofundar conhecimento na base de comandos do Python (input, if, elif, else,while, for).
2. Aprofundar meu conhecimento sobre conceito de lista, dicionários e tuplas.
3. Criar funções.
4. Consumir dados de API.

## Aonde pretendo chegar?

Neste curso pretendo fortalecer o que já aplico no dia a dia pra então seguir no estudo de algumas bibliotecas espefícas na área de dados como: Pandas, NumPy, Streamlit, Pyspark, Pyodbc, Matplotlib, Seaborn.

Em paralelo também utilizar bibliotecas que ajudem no consumo de API e raspagem de dados, automação de rotinas como Selenium e Requests.
