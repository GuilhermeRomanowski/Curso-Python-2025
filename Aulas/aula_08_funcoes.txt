- sum() 
- max()

2. Para criar uma função no python, temos que definir ela através do def ()
    - Assim como na matemática temos as funções: f(x) = x + 1.
    - No python podemos definir funções. Essa mesma função matemática ficaria assim em python:
        def f(x):
          return 1 + x
