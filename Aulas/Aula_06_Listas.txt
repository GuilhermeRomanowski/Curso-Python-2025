Nesta aula será ensinado o conceito de Listas no Pyhton.

Listas: Conjunto de elementos.
Ela fica entre colchetes [] e cada elemento é separado por ",".
Listas é formada por elementos[1,2,3,4,5]
Para exiber o elemento 5 da lista, o python interpreta como índice, iniciando a contagem do número 0
Assim a lista acima são interpretadas da seguinte maneira:

Elementos: 1, 2, 3, 4, 5
Índice: 0, 1, 2, 3, 4

Para exibir o 5º elemento da lista, devemos colocar o número do índice (4)
